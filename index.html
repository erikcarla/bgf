Welcome to BGF 2013
(BNCC Game Fest) website!

BNCC Game Fest (BGF) is the first game festival in Bina Nusantara University, organized by Bina Nusantara Computer Club (BNCC). This BGF 2013 with theme : “Your Only Gaming Zone”, cooperates with Garena Indonesia as its main sponsor. The purpose of this “Your Only Gaming Zone” here means that BGF 2013 will create a game atmosphere for all participants by presenting multiplatform game on various media. To enlighten this big event, we gladly invite all game publishers, game developers, game peripherals, IT companies, incubators and game fans to come and join us here.

This event will be on April 1-6, 2013 in Binus University and presenting : IT Exhibition, Game Developer Gathering, seminars, workshop, and game competition as its main programs. So please wait us and keep in tune for more updates.
